# Bash Setup
Setup script for configuring a bash console. Configuration files are supplied in [/config](./config).

## Installs
- generic pkgs
  - curl
  - wget
  - git
  - unzip
  - build-essential
  - ruby
  - ruby-dev
  - ruby-colorize
- applications
  - micro
  - ranger
  - colorls
  - batcat
  - neofetch
- fonts
  - nerdfont: SourceCodePro

## Configures
- ranger: ~/.config/ranger/rc.conf
- micro: settings.json
- oh-my-posh: armck.omp.json
- ~/.bashrc
